# Cinema Models Implementation Plan (US-10 — M2 first US)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (inline) to implement task-by-task. Subagent-driven execution is NOT compatible with this workflow — see role-division note below. Steps use checkbox (`- [ ]`) syntax for tracking.
>
> **Project-specific role division (CRITICAL):** This repo has an explicit rule (see `memory/feedback_role_division.md`) — **user writes ALL application code** (`.py`, `.html`, settings edits, `poetry add` calls, `makemigrations` commands); **Claude writes ALL tests** (`test_*.py`, `conftest.py`, `factories.py`). User runs ALL `git`/`gh` commands. App code in this plan is a **reference implementation** for the user to study/adapt while typing — not for paste. Test code is **complete and ready to paste** by Claude.

**Goal:** Implement six cinema models (Genre, Actor, Director, Hall, Movie, Screening) with migrations, Pillow-backed ImageField support, MEDIA config, and factory_boy factories. Closes US-10 and unblocks US-11..US-17 (M2 views, admin, seed extension).

**Architecture:** Single `apps/cinema/models.py` containing all 6 models. Pillow added as production dep; `MEDIA_URL`/`MEDIA_ROOT` configured in `settings/base.py` with dev-only static serving in `settings/urls.py`. Per-model TDD cycles with incremental migrations (`0001` through `0006`) — one migration per model commit, keeping each commit self-contained. Factories in `tests/cinema/factories.py` grow alongside models.

**Tech Stack:** Django 6 (ORM, ImageField via Pillow), pytest-django, factory_boy + Faker (`pl_PL` locale), Decimal for prices, `timezone.now()` for time arithmetic.

**Branch:** `feat/FR-3.2-cinema-models` (per backlog US-10).

```python
class Screening(models.Model):
    movie = models.ForeignKey(
        Movie,
        on_delete=models.CASCADE,
        related_name="screenings",
        verbose_name=_("movie"),
    )
    hall = models.ForeignKey(
        Hall,
        on_delete=models.PROTECT,
        verbose_name=_("hall"),
    )
    start_time = models.DateTimeField(_("start time"))
    price = models.DecimalField(
        _("price"),
        max_digits=6,
        decimal_places=2,
        validators=[MinValueValidator(Decimal("0.01"))],
    )

    class Meta:
        verbose_name = _("screening")
        verbose_name_plural = _("screenings")
        ordering = ["start_time"]

    def __str__(self) -> str:
        return f"{self.movie.title} @ {self.start_time:%Y-%m-%d %H:%M}"

    def booked_seats_count(self) -> int:
        # US-18 will sum seats_count from CONFIRMED bookings; until then no bookings exist.
        return 0

    def available_seats_count(self) -> int:
        return self.hall.capacity - self.booked_seats_count()

    def is_in_past(self) -> bool:
        return self.start_time <= timezone.now()

    def is_available(self) -> bool:
        return not self.is_in_past() and self.available_seats_count() > 0
```

## Out-of-band notes

- **Why 6 migrations instead of 1 squashed?** TDD discipline — each model commit is self-contained and the test suite goes red→green per model. Migration squashing is possible (`makemigrations --squash`) but creates a synthetic history that obscures the build order. For an educational project we keep honest history.
- **`from __future__ import annotations` in factories.py** — keeps forward references friendly (e.g., the `Genre`/`Actor`/etc. imports at top of file). Mirrors `tests/accounts/factories.py`.
- **`django_get_or_create` on Genre/Hall** — Sequence-based names guarantee uniqueness within a test, but if a test creates `GenreFactory(name="Drama")` twice (legit pattern for "use the same shared genre"), `get_or_create` semantics avoid `IntegrityError`. Same for `HallFactory(name="Sala 1")`.
- **MovieFactory M2M `extracted` shape** — caller passes an iterable: `MovieFactory(genres=[g1, g2])`. Empty `[]` or missing kwarg leaves the M2M empty. No automatic random attaching — explicit is better than implicit.
- **`Faker("date_this_year")` on MovieFactory** — returns a `datetime.date` in the current year. Acceptable for tests; if a test needs a specific release_date, override (e.g., `MovieFactory(release_date=date(2010, 7, 16))`).
- **Why `dt.timedelta(days=7)` for default ScreeningFactory.start_time?** — far enough in the future that `is_in_past()` is reliably `False` even with clock skew, but close enough that nothing about timezone rollover matters. Tests needing past/future explicitly override.
- **Pillow on Windows** — Python 3.13 + Pillow 11 has wheels; no native compilation needed locally. CI Linux runners likewise prebuilt. If `poetry add` fails on a Windows machine, falling back to `pillow>=10,<11` (older but stable) is acceptable.
- **`ImageField` empty checks** — `actor.photo.name == ""` is the canonical "no upload" assertion in Django (the `FieldFile` wrapper always exists; its `.name` is empty string when no file is attached).
